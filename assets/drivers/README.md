# Animation driving clips

These short clips provide the *motion* that the Animate feature (TPSMM) transfers
onto an uploaded portrait. The person shown is **not** the person animated — only
the relative facial motion is used.

## `subtle.mp4`

- 256×256, ~4s, silent.
- Derived from the Thin-Plate-Spline Motion Model demo driving assets, which are
  based on the **VoxCeleb** dataset.
